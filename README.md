# Dakota Concrete Resurfacing

Conversion-focused static website for Dakota Concrete Resurfacing, serving homeowners and businesses throughout the Twin Cities.

## Included

- Responsive one-page website
- Service, process, comparison, service-area and estimate sections
- LocalBusiness structured data
- Open Graph social preview
- Sitemap and robots file
- GitHub Pages workflow
- Accessible mobile navigation and reduced-motion support

## Local preview

```bash
python3 -m http.server 8080
```

Open `http://localhost:8080`.

## GitHub Pages

1. Open repository **Settings → Pages**.
2. Under **Build and deployment**, choose **GitHub Actions**.
3. Push to `main`; the included workflow deploys the site.
4. Add a custom domain only after DNS is configured.
